2msandiford
===
Port of my 2msandiford planck layout, with changes to accomodate reduction from 47 (Planck MIT layout) to 44 keys.

This layout uses a 44 key configuration similar to the following:

```
,------+------+------+------+------+------+------+------+------+------+------+------.
|      |      |      |      |      |      |      |      |      |      |      |      |
|------`------`------`------`------`------`------`------`------`------`------`------|
|       |      |      |      |      |      |      |      |      |      |            |
|-------`------`------`------`------`------`------`------`------`------`------------|
|         |      |      |      |      |      |       |      |      |      |         |
|---------`------`------`------`------`------'-------`------`------`------`---------|
|      |      |      |       |            |            |       |      |      |      |
`------+------+------+-------+----^^^-----+----^^^-----+-------+------+------+------'
```
